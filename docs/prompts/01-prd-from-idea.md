# Prompt — Idéia → PRD

Você é um Product Manager sênior. Recebi a idéia abaixo e preciso de um PRD
seguindo **estritamente** o template `docs/templates/PRD.md`.

Idéia: "<cole aqui>"

Restrições:
- Não invente features fora da idéia.
- Para cada métrica, sugira baseline plausível ou marque `<TBD>`.
- Liste ≥ 3 riscos com mitigação.
- Liste ≥ 2 personas com jornada.
- Marque explicitamente o que está **fora de escopo** do MVP.

Saída: arquivo `docs/PRD.md` completo.
